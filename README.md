# TED gear

A gear + dropdown menu component that is commonly used on TED's web
applications.

![TED-gear](https://cv-screenshots.s3.amazonaws.com/TalkstarFrontend_2015-09-11_15-23-28)

## Installation

```
ember install ember-ted-gear
```

## Usage

```hbs
<ted-gear>
  <ted-gear-option>
     An option here
  </ted-gear-option>

  <ted-gear-divider />

  <ted-gear-option>
     An option here
  </ted-gear-option>
</ted-gear>
```
